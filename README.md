# JavaScript Practice Repository

This repository contains my practice code while learning **JavaScript fundamentals to advanced concepts**.  
It includes small projects, examples, and exercises written in **HTML, CSS, and JavaScript**.  

I will keep adding new files (e.g., `variables.js`, `dataType.js`, etc.) as I cover each concept.  
Under each section, I’ll write detailed notes and examples.

---

## 📂 Repo Structure

- `index.html` → Base HTML file to test JS code
- `style.css` → Styling for HTML (optional, for small UI)
- `*.js` → JavaScript files organized by topics

---

## 📖 Topics Covered

### 1. Variables
- `variables.js`
- Declaring variables (`var`, `let`, `const`)
- Naming conventions
- Scope (function scope, block scope)
- Hoisting

### 2. Data Types
- `dataType.js`
- Primitive types (String, Number, Boolean, Null, Undefined, Symbol, BigInt)
- Reference types (Objects, Arrays, Functions)
- Type conversion & coercion
- `typeof` operator

### 3. Operators
- Arithmetic, Assignment, Comparison
- Logical operators
- Ternary operator

### 4. Control Flow
- `if`, `else if`, `else`
- `switch` statement
- Loops (`for`, `while`, `do...while`, `for...of`, `for...in`)

### 5. Functions
- Function declaration & expression
- Arrow functions
- Parameters & arguments
- Default parameters
- Return values
- Higher-order functions
- Callback functions

### 6. Objects
- Object literals
- Accessing & modifying properties
- `this` keyword
- Object methods
- Constructor functions & `new` keyword
- `Object.keys()`, `Object.values()`, `Object.entries()`

### 7. Arrays
- Creating arrays
- Common methods (`push`, `pop`, `shift`, `unshift`, `map`, `filter`, `reduce`, etc.)
- Iteration techniques

### 8. DOM Manipulation
- Selecting elements (`getElementById`, `querySelector`)
- Changing content & styles
- Event listeners
- Creating & removing elements

### 9. Advanced JavaScript
- Closures
- Prototypes & Inheritance
- ES6+ features (Destructuring, Spread, Rest, Template literals, etc.)
- Promises & Async/Await
- Modules (`import`/`export`)
